Built-in commands

Four slash commands cover the entire workflow

Open this repo in Claude Code and these commands are immediately available.

/new-skill
Guided wizard. Asks the right questions and creates a well-structured SKILL.md for you.
/export-to-sdk
Generates production-ready Python Agent SDK code from your skill. Ship your MVP before you harden it.
/eval-skill
Runs your test cases, grades each output using Claude-as-judge, and writes a pass/fail report.
/improve-skill
Reads the eval report and directly edits your SKILL.md to fix failures — takes effect in production immediately.

The mental model

A Skill is a job description for your AI

Skill              =   A job description for your AI (plain markdown)
Agent SDK query    =   That job running autonomously in production
ClaudeAgentOptions =   Permissions + tools + configuration

A Skill is just a SKILL.md file with YAML frontmatter and step-by-step instructions. Claude Code reads it when you invoke the command and uses it as instructions. The same content becomes your prompt in the Agent SDK — so what you tune during prototyping is exactly what runs in production.
